个人习惯使用 vim 操作模式，vscode、obsidian 等软件都有 vim 模式选项，浏览器有 Vimium C 插件，为了使用顺手需要记住/更改/新加一些快捷键  
**Vimium C**  
自定义快捷键：

```
unmap o
unmap O
map <a-j> previousTab
map <a-k> nextTab
map <a-l> goForward
map <a-h> goBack
unmap J
unmap K
map t LinkHints.activateEdit
```

定义了 alt+j/k 切换浏览器窗口  
alt + h/l 返回本窗口的上/下一个页面  
t 选择输入框

**vscode**  
设置 alt+j/k 切换窗口

**vim**  
`V`选择几行后，按`<`/`>`选择取消缩进/缩进
