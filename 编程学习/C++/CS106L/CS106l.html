manipulator ​

  • setw(set width)用来控制cout输出的宽度,头文件<iomanip>
cpp
cout << setw(10) << 111 << endl;
1

还可以控制是左空白还是右空白：

cpp
cout << '[' << left << setw(10) << "Hello!" << ']' << endl;
cout << '[' << right << setw(10) << "Hello!" << ']' << endl;
1
